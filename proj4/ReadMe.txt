PROJECT TITLE: Poker
PURPOSE OF PROJECT: To play a pseudo game of poker. The the purpose of the algorithm is to supply 
player with five cards, define and score the hand player receives in a format
understandable by commong poker players, exchange cards if the desires, score 
the hand one final time and ask user if he wants to play again.
VERSION or DATE: First version, 11/9/11
HOW TO START THIS PROJECT: Define the steps it takes for the common poker player to accept do the steps 
metioned in, "the purpose of the project." Write the methods that are similar to the actions taken
throughout a regular poker game. Draw up constructors and classes respectively.
